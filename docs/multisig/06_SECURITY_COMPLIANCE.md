
# Security & Compliance Positioning

## Key Positioning

CoinPay is:
- A dispute mediator
- A multisig co-signer
- Not a custodian

## Important Rules

- No unilateral execution
- No fiat custody
- Arbiter key stored in HSM
- Full audit logs
- No pooled funds

## Messaging

"Non-custodial escrow secured via multisignature."
